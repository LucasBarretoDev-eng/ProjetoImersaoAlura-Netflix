🎬 Netflix Clone - Imersão Alura (IA Edition)
Projeto desenvolvido durante a Imersão Alura, com o objetivo de recriar a interface da Netflix e explorar o potencial das Inteligências Artificiais no auxílio ao desenvolvimento de software, acelerando a escrita de código e a resolução de problemas.

🚀 Tecnologias Utilizadas
HTML5 e CSS3 para a estrutura e estilização da interface.

JavaScript para a manipulação do DOM e dinamismo da página.

Ferramentas de IA: Utilizadas para otimização de código, geração de ideias de design e auxílio na correção de bugs.

Git & GitHub: Para controle de versão e hospedagem do código.

🧠 O que eu aprendi
Este projeto foi além do desenvolvimento web tradicional. Os principais aprendizados foram:

Prompt Engineering: Como fazer as perguntas certas para IAs para obter trechos de código funcionais e eficientes.

Refatoração assistida: Uso de IA para melhorar a legibilidade do código CSS e JS.

Manipulação de Elementos: Criação de carrosséis e efeitos de hover dinâmicos.

Integração IA + Dev: Entender como a IA atua como um "copiloto", permitindo focar na arquitetura do projeto enquanto ela auxilia em tarefas repetitivas.
